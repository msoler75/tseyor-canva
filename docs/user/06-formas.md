# Formas

## Añadir una forma

1. Desde InsertSidebar, seleccionar "Forma"
2. Elegir tipo: rectángulo, círculo, triángulo, etc.
3. La forma se añade a la página activa

## Propiedades editables

- **Relleno**: color sólido o degradado
- **Borde**: color, grosor, estilo
- **Opacidad**
- **Rotación**
- **Efectos**: neón, sombra, contorno
- **Redimensionamiento**: permite salir de la página

## Degradados

El editor incluye paletas de degradados predefinidas y editor de colores personalizado.
