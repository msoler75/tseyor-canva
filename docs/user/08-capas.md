# Capas

## Orden Z

Cada elemento tiene una posición en el eje Z (profundidad) que determina qué elementos están delante o detrás de otros.

## Panel de capas

Desde InsertSidebar → pestaña "Capas":
- Lista de todos los elementos de la página activa
- Arrastrar para reordenar
- Click para seleccionar
- Botones: traer al frente, enviar al fondo, subir/bajar capa

## Comportamiento

- Elementos con mayor zIndex se muestran delante
- Elementos con menor zIndex detrás
- Se pueden apilar elementos para crear composiciones
