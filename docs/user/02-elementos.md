# Elementos

## Tipos de elementos

| Tipo | Descripción |
|------|-------------|
| Texto normal | Título, subtítulo, texto plano (formato uniforme por caja) |
| Texto enlazable | Caja de texto enriquecido que puede enlazarse en secuencia |
| Imagen | Imagen subida por el usuario |
| Forma | Figuras predefinidas (rectángulo, círculo, triángulo, etc.) |
| Código QR | QR generado a partir de datos |

## Selección

- Click simple: selecciona el elemento y muestra sus handles de redimensionamiento
- Click en página vacía: deselecciona todo
- Shift+click: añade/quitar de la selección múltiple

## Manipulación

- **Mover**: arrastrar desde el interior del elemento
- **Redimensionar**: arrastrar desde los handles (esquinas/bordes)
- **Eliminar**: seleccionar y presionar Supr o Delete
- **Duplicar**: seleccionar y presionar Ctrl+D
- El redimensionado y movimiento puede salir de los límites de la página

## Propiedades

Cada elemento tiene propiedades editables desde el panel contextual:
- Posición (x, y)
- Tamaño (w, h)
- Rotación
- Opacidad
- Efectos (neón, sombra, contorno)
- Borde
- Orden Z (capa)
