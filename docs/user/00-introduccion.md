# Introducción

TSEYOR Canva es un editor de diseño visual que permite crear carteles, folletos, dípticos, trípticos y otros documentos gráficos directamente desde el navegador.

## Funcionalidades principales

- Editor visual con lienzo, textos, imágenes, formas y código QR
- Asistente de diseño para guiar la creación paso a paso
- Texto enlazable: flujo de texto entre múltiples cajas (ideal para folletos)
- Plantillas reutilizables clasificadas por categoría y objetivo
- Historial undo/redo
- Autoguardado
- Exportación a imagen
- Soporte multipágina (hasta 255 páginas)

## Acceso

La aplicación requiere inicio de sesión mediante el portal TSEYOR. Los usuarios invitados pueden crear diseños temporales que se convierten en permanentes al iniciar sesión.
