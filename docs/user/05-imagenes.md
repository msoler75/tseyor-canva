# Imágenes

## Subir una imagen

1. Desde el panel InsertSidebar, seleccionar "Imagen"
2. Elegir archivo del equipo
3. La imagen se añade a la página activa

## Edición

- **Redimensionar**: arrastrar handles (mantiene proporción)
- **Mover**: arrastrar
- **Rotar**: desde panel de propiedades
- **Opacidad**: ajustable
- **Efectos**: neón, sombra, contorno

## Formatos soportados

Formatos de imagen comunes (JPG, PNG, WebP, GIF). Las imágenes se procesan con límites configurables (ancho/alto máximo 2400px por defecto).

## Comportamiento

- Se permite redimensionar/desplazar imágenes fuera de la página
- Al exportar, se renderizan con su opacidad y efectos aplicados
