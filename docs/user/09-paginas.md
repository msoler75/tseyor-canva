# Páginas

## Multipágina

Los diseños pueden tener múltiples páginas (hasta 255). Especialmente útil para:
- Folletos dípticos (2 páginas)
- Folletos trípticos (3 páginas)
- Documentos extensos

## Navegación

- Scroll vertical para cambiar de página
- Click en una página para hacerla activa
- La página activa tiene un borde destacado
- Las acciones de inserción se aplican a la página activa

## Páginas en folleto

En formato folleto, cada página física tiene dos paneles (izquierdo y derecho) que representan dos páginas consecutivas en el flujo de lectura.

## Duplicar página

- Se duplica el contenido completo de la página
- Los textos enlazables se convierten en independientes

## Añadir/eliminar páginas

Desde los controles de página en el editor.
