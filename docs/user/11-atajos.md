# Atajos de teclado

## Generales

| Atajo | Acción |
|-------|--------|
| Ctrl+Z | Undo |
| Ctrl+Shift+Z / Ctrl+Y | Redo |
| Ctrl+S | Guardar |
| Ctrl+D | Duplicar elemento |
| Supr / Delete | Eliminar elemento |
| Escape | Cancelar edición / deseleccionar |

## Editor de texto

| Atajo | Acción |
|-------|--------|
| Ctrl+B | Bold/negrita |
| Ctrl+I | Cursiva |
| Ctrl+U | Subrayado |
| Ctrl+V | Pegar (con diálogo de formato en linkedText) |

## Navegación

| Atajo | Acción |
|-------|--------|
| Click | Seleccionar elemento |
| Doble click | Editar texto |
| Arrastrar | Mover elemento |
| Arrastrar handle | Redimensionar |
| Shift+click | Selección múltiple |
